## Usage

1. Create and env
    ```bash
    python -m venv env
    ```
2. Activate env
    ```bash
    source env/scripts/activate
    ```
3. Install from `requirements.txt`
    ```bash
    pip install -r requirements.txt
    ```
4. Run the program
   ```bash
   python main.py
   ```
